# An-lise-de-Faturamento-e-Performance-Comercial-Excel-

Transformei uma base simples de dados em um pequeno estudo de performance comercial, usando Excel e tabelas dinâmicas.
Ao invés de olhar apenas valores isolados, a ideia foi cruzar métricas para entender melhor o desempenho dos funcionários.

- O que foi analisado:
Faturamento por funcionário
Comparativo entre faturamento e meta
Ranking de % de atingimento da meta

- Alguns insights que emergem da análise:
Alto faturamento não significa, necessariamente, melhor desempenho em relação à meta
O % de atingimento ajuda a equilibrar a leitura entre volume e eficiência
Comparar faturamento x meta traz mais clareza do que analisar apenas valores absolutos

- Ferramentas utilizadas:
Excel online
Tabelas dinâmicas
Gráficos comparativos
Formatação condicional para facilitar a leitura
Esse projeto começou como um exercício simples e acabou se tornando uma análise bem mais completa do que eu imaginava  um ótimo exemplo de como organização e visualização de dados fazem diferença na tomada de decisão
